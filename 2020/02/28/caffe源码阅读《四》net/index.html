caffe的net是一个网络架构，串联起所有的blob，支撑着整个神经网络前向和反向传播的一个结构。他也是分为hpp和cpp文件的。
首先看注释

把各个层连接在一起，形成一个有向无环图。
接着直接看代码

Net 构造函数

首先是构造函数，他又两个，第一个传入的是 NetParameter的对象，第二个是通过文件读取的，传入的文件路径和一个 phase, phase是指的我这个网络的状态是 train 还是 test的，然后还有 level 和 stages.
看看它的具体实现

由于我们参数是从文件读取的，所以这个

是我们自己定义的。
然后是执行了

这个函数传入了两个参数，一个是这个文件的地址，一个是 param这个变量，用来接收文件中读取的配置参数。这个函数是位于 src/caffe/util/upgrade_proto.cpp这个目录下的，它的大概功能就是从这个文件里面读取参数到 param这个变量里去。这个 param的格式是在 src/caffe/proto/caffe.proto里面定义的，我们可以去看一看。

这里面还是定义了很多东西的，并且都有非常详细的注释，比方说网络的名字，形状，是否打印等等。
紧接着是设置状态，就是下面这一步。

然后如果设置了 stage的话，就把每一层的 stage加载进来。
然后 level也是同理

最后初始化一下 param就完了

然后我们再看直接传入这个 NetParameter的构造函数

他就是直接调用了 Init这个函数。
然后我们再看一下 Init这个函数，它也是在这个 Net类里面。

Init 函数

这个函数实际上就是做一个初始化的功能。

比方这一句就是把从参数里读取的 phase赋值给自己的私有变量 phase_.

然后定义了一个 NetParameter的变量，去调用了一个 FilterNet函数， 这个FilterNet函数是干嘛的呢？
它是根据 state初始化所有层的一个函数。这个后面紧跟着就会讲解。

然后跟着的是 InsertSplits这个函数，这个函数是干嘛的呢？
因为有的层可能会分裂，下面会有两条路可以走，比方说有两个top，那么这个时候就需要分割出来了。

接下来的这个是根据layer的大小给这些vector分配空间

给每个layer分配 phase，决定它是 train还是 test

然后开始把每个layer链接起来，因为它是自 top向 bottom链接的，所以这里是 AppendBottom，他的 bottom添加了对应的blob，也就是相当于一个链表一样的结构，每个 bottom添加了后面一个 bottom的id。

然后是把 top对应层的id添加进去，也就是相当于一个双向的链表一样的。

如果你这个层定义了自动top的话，那么这里就会把自动top添加进去。
这个自动top blob就是之前我们在layer看到的这么一个接口。至于每个层具体怎么实现的其实是在不同层有不同的实现方法的。

全部都添加好了以后就开始走这个层的 setup函数，这个 setup我们之前在 layer里面讲到过，就是那4步，就是。

  • 检查blob的count
  • 走layer的setup
  • 然后reshape它
  • 最后设置loss权重 然后resize一下loss权重的大小，最后还统计了一下会用到多少内存，然后打印出来。 接着设置了每一层的 propagate_down 再把每一层的参数添加进去。 最后设置一下反向传播的状态，如果参数里指定了这一层有反向传播那么这里就会设置成 true
    这样我们这个对于layer初始化的过程就完成了 然后他说通过网络反向传播去决定哪一个blob贡献了loss，就可以跳过这些不需要贡献loss的反传计算。
    然后看看是不是所有的blob都不需要反传，如果是的话那么我们就可以直接跳过 propagate_down这一步了 首先默认他没有贡献loss的。 如果发现它贡献了loss就把这个 layer_contributes_loss设置成 true 如果一直没有做贡献的话那么我们就可以跳过 propagate_down这一步了。 然后这里处理了一下强制反传，就是设置一下哪些blob需要强制反传，哪些不需要。 然后又弄了一个做 net_output_blobs_的vector，就相当于把这个网络串联起来，类似于一个单链表结构。 接着从这个参数里面抄权值，然后设置一下debug的状态。
    最后这个 Init函数就结束了。
    这样我们构造函数和 Init函数就结束了。

FilterNet 函数

这个函数是根据 state初始化所有层的一个函数。state包含了 phase, level stage

这里拷贝了参数到 param_filtered这个变量里面

然后调用 clear_layer清除所有层，因为它是刚刚开始初始化时候调用的，就需要把内存里的一些垃圾信息删掉。

这里循环遍历所有层，然后
加载每一层的参数和名字

检查每一层的 StateMeetsRule

这个StateMeetsRule属于级联网络才用得到的开关，我们先跳过这一步。
然后开始copy layer

之前不是把layer都clear了吗，现在就把从param里读的参数添加给layer。

Forward 函数

这个是 Net的 Forward函数，不是 Layer的Forward函数，Net的 Forward函数是指挥每一个 Layer往哪个地方去传。

这里判断了一下需不需要 loss,如果不需要 loss就
直接走这个 ForwardFromTo这个函数

ForwardFromTo 函数

我们可以进来看一下

首先让 before_forward_的这些步骤先走，因为他们的名字就是在前传之前的步骤，他们走完之后就开始走 layer的步骤

然后 layer挨个走完 Forward之后，有个 loss把每一步额的 loss加起来。
这个走完之后就是 after_forward

也是顾名思义，所以他们在 forward之后也走了一遍。
最后把总的 loss返回就可以了。

因为这个是net的 Forward函数，他就是一个大的框架，起的是串联的作用，所以比较简单。

ForwardPrefilled 函数

他的注释上写了,使用 Forward代替

而且他的实现也就是直接调用的 Forward函数，所以这个函数实际上是一个比较老的函数，他只是留下了一个接口而已，以后都不会再用到了。

ForwardFromTo 函数

这个函数注释上就写的很明白了，就是指定从某一层到某一层的一个函数，与之类似的还有 ForwardFrom ForwardTo。都只是参数不同而已。

Forward 函数

这里是另外的一个 Forward函数

可以看到，他把 loss放在了参数里面,这个也是一个比较老的实现方法了，现在也都是用之前哪个 Forward函数来实现了。

以前是会把 bottom都传进来的，而现在 bottom是在初始化的时候就加载了，所以不用传了。但是保留了原来的接口，所以主页里还是先复制一下它传进来的 bottom，然后再 Forward

ClearParamDiffs 函数

他说了，这里是把所有的参数都清成0，这个必须得在反传之前运行。

Backward 函数

反传函数其实和前传是一样的，因为这个是net的反传，它只是一个架构的作用

可以看到，它第一步就是调用这个 BackwardFromTo函数，实际上和前传是一样的。

然后下面这些都是需要debug的一些参数，比方说各个data，各个diff，还有第二范式等等。

BackwardFromTo 函数

这个 BackwardFromTo函数和前传是一样的，设置一个起点，设置一个终点，然后运行 before_backward_，完了运行 Backward，最后运行 after_backward_

Reshape 函数

这里是net的Reshape，不是blob的Reshape。这里说Reshape会改变所有的layer的 top bottom的shape，也就是说你你前传的时候如果就没有改变这个layer的大小的话就会调用这个函数。

可以看到，它的实现非常简单，就是遍历了所有layer，每个layer都Reshape了一下。
因为layer的Reshape是调整了一下top的形状，然后给他分配缓存空间。

ForwardBackward 函数

这个函数就是 Forward和 Backward结合起来

最后他也会返回一个前传得到的loss，这一步是实现好了的，可以方便的在训练时候去调它。

Update 函数

这个 Updata函数是根据计算的diff来更新网络的权重

它调用的是 learnable_params_的 Update,而这个learnable_params_的 Update是 Blob里的函数，他就是计算权重的一个函数。

ShareWeights 函数

这个函数是从参数里面读取那些权重，这个函数只在 Net::Init里调一次，所以不应该手动去调用它。

ShareTrainedLayersWith 函数

它的注释是说从已经初始化了的网络，隐式的去拷贝，不占用内存，也就内存映射的意思，其实就是加载一个pretrain的模型。

可以看到，它的指针指向了传进来的那个layer，这个意思就是内存映射的意思。

copy 函数

这些都是拷贝的函数，就是从文件里面拷贝一个网络到对象里面。

toProto 函数

这些是把网络储存到文件中的两个函数，一个是proto一个是HDF5格式

剩下的函数都是返回成员变量的一些函数了，比方说返回名字等等。
